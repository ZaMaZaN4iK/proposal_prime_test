Overview

Programmers sometimes need to check whether the number is prime or not. This task is especially common in cryptography. This proposal suggests extending the set of mathematical functions with functions to check whether the number is prime or not.

Design overview

This paper proposes two prime-test functions.

Header <cmath> synopsis

namespace std {
namespace experimental {

bool is_prime(Integral value) noexcept;

bool is_probable_prime(Integral value, Integral certainty) noexcept;

} /* namespace experimental */
} /* namespace std */
is_prime 
bool is_prime(Integral value) noexcept;

The function tests is a value prime or not. If a value is prime, returns true. Otherwise, false.

Complexity: O(sqrt(N)).

is_probable_prime 
bool is_probable_prime(Integral value, Integral certainty) noexcept;
Parameters:

certainty - a measure of the uncertainty that the caller is willing to tolerate: if the call returns true the probability that value is prime exceeds (1 - 1/2^certainty).

Returns true if value is probably prime, false if it's definitely composite. If certainty is <= 0, true is returned.
